# QuizBattle Client

Фронтенд для realtime-квиз платформы на React + Vite + TypeScript.

## Команды

```bash
npm run dev
npm run build
npm run lint
```

## Точки интеграции

- Socket.IO сервер: `VITE_SERVER_URL` (по умолчанию `http://localhost:4000`).
- Рендерит лобби, матч и аналитику.
- Показывает статус соединения и live-состояние комнаты.
